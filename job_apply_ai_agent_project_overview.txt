üí° PROJECT NAME:
Job Apply AI Agent

üìù PROJECT DESCRIPTION:
An intelligent AI-powered tool that automates the job application process. It:
- Scrapes job listings (e.g., from LinkedIn)
- Analyzes job descriptions using NLP to extract requirements
- Tailors your CV and generates customized cover letters
- Supports batch processing for multiple applications

=================================================[]

üöÄ KEY FEATURES:

1. **Job Scraping**
   - Scrapes latest job postings based on role, location, etc.
   - Removes duplicate job listings using smart filtering.

2. **NLP-Based JD Analysis**
   - Extracts skills, keywords, and job responsibilities.
   - Matches them against your existing CV content.

3. **CV Customization**
   - Adjusts your CV dynamically with skills and keywords relevant to each job.
   - Respects the original document structure (DOCX/PDF).

4. **Cover Letter Generation**
   - Auto-writes personalized cover letters based on job and company information.

5. **Batch Processing**
   - Processes multiple jobs at once.
   - Outputs custom CVs + cover letters for each role.

6. **User Interface Options**
   - Easy-to-use Web UI (Upload -> Process -> Download)
   - CLI mode for batch automation.

=================================================[]

üß† HOW IT WORKS (Pipeline)

1. **Input your base CV (docx/pdf).**
2. **Define target job search criteria** (keywords, titles, locations).
3. **Job agent scrapes relevant listings.**
4. **NLP agent analyzes job descriptions.**
5. **Keyword matching & CV tailoring.**
6. **Cover letter generation with ATS-friendly formatting.**
7. **Output**: New folders with tailored application materials.
